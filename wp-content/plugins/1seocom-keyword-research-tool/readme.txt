== Description ==

WordPress keyword research tool.  This plugin will help you find potential keywords to optimize for your website. It is currently using Google API to fetch the auto suggested words that people are searching for on the search engine. 

The suggestions that Google provides come from what people are searching.  

Example if you typed in **seo services**:

* seo services pricing
* seo services company
* seo services reviews

These searches are real and done by other people.  The popularity of these keywords are a factor of what Google will show.

== Installation ==

1. Upload `1seokwr` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin will appear on the side menu labeled as "1SEO.com KWR"

== Frequently Asked Questions ==


= Are there any limitations to this plugin? =

No limitations are set, but Google might temporarily stop you from using the plugin if you are making too many requests.  This block usually expires within 15 minutes to 30 minutes.
